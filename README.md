## Basic Concept
##### 1. How to create a new nuxt 3 app?
Open terminal and run this command
```
npx nuxi@latest init <project-name>  
```
If face npm error, run this command
```
npm install npm -g
npx nuxi@latest init <project-name>  
```
After successfully create project then run command
```
cd project-name
npm run dev
```
